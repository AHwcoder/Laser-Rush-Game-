# Laser Rush 

An exciting, fast-paced First-Person Shooter (FPS) game developed exclusively for **PC** and designed specifically for children aged **8 to 12 years**. In a vibrant futuristic world, players take on the role of the brave hero **"Barhoum" (برهوم)** as he faces waves of intelligent robotic drones in a thrilling sci-fi arena.

This project showcases a complete Software Development Life Cycle (SDLC), demonstrating advanced concepts in Game Design, Artificial Intelligence, Quality Assurance, and Accessibility.

---

##  Key Features

* **Cross-Input Compatibility (PC):** Fully compatible with both **Keyboard & Mouse** and **Gamepads/Controllers** (Plug-and-Play), featuring a custom unified `PlayerInputHandler` script.
* **Tailored Controls:** Input sensitivity is carefully calibrated and optimized for both analog sticks and mouse movement to ensure smooth, accessible handling for younger players.
* **Engaging AI Navigation:** Enemy robots utilize advanced pathfinding algorithms to track Barhoum and intelligently navigate around obstacles and walls.
* **Performance Optimized:** Highly optimized architecture running smoothly at a stable **75+ FPS** on standard PC hardware.
* **Inclusive Design (Accessibility):** Features a dedicated High-Contrast (Black & White) visual mode to assist visually impaired players.
* **Polished Gameplay Mechanics:** Includes a custom script-driven HUD with a dynamic countdown timer (`00:00` format) to enhance the challenge.

---

## 📁 Repository Structure

This repository is organized into two main components:
* `Laser-Rush-Build/`: Contains the standalone, ready-to-play PC executable file (`My game FPS.exe`) because this the game First Personal Shotter Mode . **No installation or Unity required to run!**
* `Laser-Rush-Source/`: The root directory containing the complete Unity project, including all C# source code scripts, scenes, and assets.

---

##  Tech Stack & Tools

* **Target Platform:** PC (Windows Standalone)
* **Input System:** Integrated Keyboard/Mouse & Controller Support
* **Game Engine:** Unity (Built using the modified FPS Microgame Template under proper software licensing)
* **Language:** C#
* **IDE:** VS Code
* **Version Control & Management:** Git / GitHub & Gantt Project Management

---

##  How to Play

1. Download or clone this repository to your PC.
2. Connect your preferred input device (**Keyboard/Mouse** or **Gamepad**).
3. Navigate to the `Laser-Rush-Build/` directory.
4. Double-click **`Laser Rush.exe`** to start playing immediately!
5. Help **Barhoum** survive the laser rush!
